Windows常用排查操作

图形界面查看

mstsc 打开远程桌面连接
taskmgr 打开任务管理器
ncpa.cpl 打开网络连接面板
regedit 打开注册表编辑器
services.msc 打开服务面板
taskschd.msc 打开任务计划程序
eventvwr.msc 打开时间查看器
lusrmgr.msc 打开本地用户与组

命令行查看

查看系统基本信息

systeminfo

查看系统环境变量

set

查看已安装的补丁

wmic qfe get Caption, Description, HotFixID, InstalledOn

查看系统日志

wevtutil qe Security /f:text   查看安全日志
wevtutil qe System /f:text   查看系统日志
wevtutil qe Application /f:text   查看应用日志

查看计划任务

schtasks /query /fo LIST /v
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Schedule\TaskCache
包含几个子项：
Tasks- 包含任务定义
Tree- 包含任务的组织结构
Plain- 包含其他任务信息
Boot- 包含启动时运行的任务

查看注册表启动项

reg query HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run   查看当前用户启动项
reg query HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Run   查看所有用户启动项
reg query HKLM\Software\Wow6432Node\Microsoft\Windows\CurrentVersion\Run   查看64位或32位系统启动项

使用WMIC命令查看

wmic startup get caption, command, location, user

查看系统服务

sc query //查看所有运行中的服务
sc query [SEARVICE_NAME] //查看指定的服务
sc query state= all //查看所有状态的服务，包括停止的，禁用的
net start //仅查看已启动的服务，仅显示服务名
查看所有服务的详细列表（包括进程和命令行参数）：
wmic service get name,displayname,pathname,startmode,state

查看用户

query user  查看所有登录中的用户
net user  查看所有本地可见用户
net user /domain 查看所有域内可见用户
net localgroup administrators

该方法无法检查服务器上的隐藏账户。“隐藏账户”通常是通过在 Names键下的账户名前面添加一个 $符号来实现的（例如，将 admin隐藏为 $admin）

通过注册表检查服务器的用户

在注册表中导航到HKEY_LOCAL_MACHINE\SAM\SAM\Domains\Account\Users\Names，观察是否存在异常用户，并且对比与net user命令的返回结果是否一致。

查看进程信息

tasklist   //查看简易任务列表
wmic process get Name, ProcessId, ParentProcessId, CommandLine   //查看详细的进程信息，包含命命令行参数

查看网络连接

netstat -ano
