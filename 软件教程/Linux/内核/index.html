内核

升级内核版本

概述

Linux内核是开源的程序，使用C语言编写，其广泛安装在其他版本操作系统上，称发行版。

通过包管理工具升级

以下操作适用于Linux CentOS发行版

开启ELRepo仓库

导入公钥
rpm --import https://www.elrepo.org/RPM-GPG-KEY-elrepo.org
安装ELRepo包
yum install -y https://www.elrepo.org/elrepo-release-7.el7.elrepo.noarch.rpm

列出可用内核版本

yum --disablerepo=* --enablerepo=elrepo-kernel list kernel*

ELRepo提供了两种类型的内核：

  • kernel-lt：长期支持（LTS）内核，稳定性高，适合生产环境。
  • kernel-ml：主线（Mainline）内核，版本较新但可能不够稳定。

安装内核版本

#安装长线版本
yum --enablerepo=elrepo-kernel install kernel-lt -y
#安装主线版本
yum --enablerepo=elrepo-kernel install kernel-ml -y

设置默认启动内核

grub2-set-default 0      #设置新内核为默认启动项
grub2-mkconfig -o /boot/grub2/grub.cfg     #重新生成 grub.cfg

重启系统

reboot

通过源码编译升级

收集系统信息

查看系统版本
# cat /etc/os-release
查看内核版本
# uname -a
3.10.0-1160.88.1.el7.x86_64
查看GCC版本
# cat /proc/version

下载地址

https://mirrors.edge.kernel.org/pub/linux/kernel/v5.x/linux-5.11.2.tar.gz

tar -xf linux-5.11.2.tar.gz

cd linux-5.11.2
