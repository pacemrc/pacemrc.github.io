Screen命令

screen 是 Linux 下的终端复用与会话保持工具。它的核心价值在于：让你在断开 SSH 连接后，让任务在服务器后台继续运行，重新连接时能“原封不动”地恢复现场。

1. 会话生命周期管理

命令 说明
screen -S <name> 创建一个名为 <name> 的新会话（推荐给会话起名，便于恢复）
screen -ls 列出当前所有 screen 会话（查看有哪些后台任务）
screen -r <name/pid> 恢复/连接到指定的后台会话
screen -d <name> 分离（Detach）指定会话（使其在后台运行，不终止进程）
exit 或 Ctrl + a k 关闭当前窗口（退出会话，任务终止）

2. 会话内快捷键操作

所有快捷键均以 Ctrl + a 为前缀（先按 Ctrl + a，松开后再按下一个键）。

快捷键 功能
Ctrl + a c Create - 在当前会话内创建一个新的窗口
Ctrl + a n Next - 切换到下一个窗口
Ctrl + a p Previous - 切换到上一个窗口
Ctrl + a " 列出所有窗口，供选择切换
Ctrl + a d Detach - 分离当前会话（返回主 shell，会话在后台继续运行）
Ctrl + a k Kill - 强制杀死当前窗口
Ctrl + a [ 进入滚动模式（查看历史输出，按 Esc 退出）
