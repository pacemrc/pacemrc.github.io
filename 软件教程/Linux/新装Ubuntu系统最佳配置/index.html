新装Ubuntu系统最佳配置

1. 关闭防火墙与 SELinux

关闭防火墙

# 停止并禁用防火墙
systemctl stop ufw
systemctl disable ufw

关闭 SELinux

Ubuntu 默认没有启用 SELinux，无需配置；

2. 配置阿里云源

https://developer.aliyun.com/mirror/ubuntu

3. 修改系统时区

timedatectl set-timezone Asia/Shanghai

4. 安装常用命令

# 更新软件包索引
apt update -y
# 安装常用工具
apt install -y wget vim zip unzip tcpdump lsof net-tools bash-completion apt-utils telnet screen netcat-traditional git curl tree ffmpeg
# 安装开发工具包
apt install -y build-essential

5. 安装常用软件

安装 Docker

安装依赖

apt install -y apt-transport-https ca-certificates curl software-properties-common

创建密钥目录并下载 Docker 官方 GPG 密钥

install -m 0755 -d /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg -o /etc/apt/keyrings/docker.asc
chmod a+r /etc/apt/keyrings/docker.asc

添加 Docker 官方稳定版仓库：

echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.asc] https://download.docker.com/linux/ubuntu $(. /etc/os-release && echo "$VERSION_CODENAME") stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

安装 Docker

apt update -y
apt install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin

启动并设置开机自启

systemctl start docker
systemctl enable docker

配置国内镜像加速

# 创建配置目录
mkdir -p /etc/docker
# 写入镜像加速配置
tee /etc/docker/daemon.json <<-'EOF'
{
    "registry-mirrors": [
        "https://docker.1ms.run",
        "https://doublezonline.cloud",
        "https://dislabaiot.xyz",
        "https://docker.fxxk.dedyn.io",
        "https://dockerpull.org",
        "https://docker.unsee.tech",
        "https://hub.rat.dev",
        "https://docker.1panel.live",
        "https://docker.nastool.de",
        "https://docker.zhai.cm",
        "https://docker.5z5f.com",
        "https://a.ussh.net",
        "https://docker.udayun.com",
        "https://hub.geekery.cn"
    ]
}
EOF
# 重启 Docker 生效
systemctl daemon-reload
systemctl restart docker

安装 Java

# 安装 OpenJDK 11（常用稳定版）
apt install -y openjdk-11-jdk
# 查看版本
java -version

安装 Python

Ubuntu 20.04+ 默认自带 Python3，直接安装 pip：

apt install -y python3 python3-pip python3-venv
# 查看版本
python3 --version
pip3 --version

安装图形化界面

# 安装桌面环境依赖
apt install -y xorg
# 安装 GNOME 桌面
apt install -y ubuntu-desktop
# 启动图形界面
systemctl set-default graphical.target
reboot

6. 配置 SSH

编辑 SSH 配置文件

vim /etc/ssh/sshd_config

# 允许 root 用户登录
PermitRootLogin yes
# 允许密码登录
PasswordAuthentication yes

重启 SSH 服务生效：

systemctl restart sshd
