添加定时任务

前言

需要在mac上添加一个定时任务。第一想到的是使用crontab，后续发现当mac进入休眠模式后crontab的定时任务无法执行，后续确定原因是使用crontab -e创建 了用户级别的任务，当mac进入休眠模式后会暂停系统上的用户级任务，所以才没有执行。

文件夹

系统级定时任务

/System/Library/LaunchDaemons 不用登陆也要运行,Apple 提供的系统守护进程
/System/Library/LaunchAgents Apple提供的代理，适用于每个用户的所有用户
/Library/LaunchDaemons 不用登陆也要运行，第三方系统守护进程
/Library/LaunchAgents  需要root登录并且配置文件归root所有

用户级定时任务

~/Library/LaunchAgents 需要非root登录，并且配置文件归非root所有

在对应文件夹下创建plist文件

com.pacemrc.douyin-downloader.plist

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>Label</key>
    <string>com.pacemrc.douyin-downloader</string>    <!-- 任务的名称 -->

    <key>ProgramArguments</key>
    <array>  <!-- 程序执行参数 -->
        <string>/Library/Frameworks/Python.framework/Versions/3.12/bin/python3</string>
        <string>/Users/pacemrc/PycharmProjects/douyin-downloader/DouYinCommand.py</string>
    </array>

    <key>StartCalendarInterval</key>   <!-- 每天23:55分执行 -->
    <dict>
        <key>Hour</key>
        <integer>23</integer>
        <key>Minute</key>
        <integer>55</integer>
    </dict>

		<!-- 设置输出日志 -->
    <key>StandardOutPath</key>
    <string>/Users/pacemrc/tmp/douyin-downloader.out</string>

    <key>StandardErrorPath</key>
    <string>/Users/pacemrc/tmp/douyin-downloader.err</string>
</dict>
</plist>

修改文件权限

chown -R root:wheel /Library/LaunchDaemons/com.pacemrc.douyin-downloader.plist

卸载任务

sudo launchctl bootout system /Library/LaunchDaemons/com.pacemrc.douyin-downloader.plist

加载任务

sudo launchctl bootstrap system /Library/LaunchDaemons/com.pacemrc.douyin-downloader.plist

查看任务详情

sudo launchctl print system/com.pacemrc.douyin-downloader

立即执行定时任务

launchctl kickstart -k system/com.pacemrc.douyin-downloader
