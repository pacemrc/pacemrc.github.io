NodeJS安装

通过Nvm版本管理器安装

nvm全称Node version mannger，可以快速的切换nodejs版本。

安装nvm

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.4/install.sh | bash

查看版本

nvm -v

安装指定版本

nvm install 22

切换指定版本

nvm use 22

# 临时切换到系统版本 
nvm use system
# 永久切回系统版本
nvm alias default system

nrm源管理

安装

npm install -g nrm

查看版本

nrm -V

查看当前使用源

nrm current

查看可用源

nrm ls
# 标*号的是当前正在使用的

切换 npm 源

# 切换到淘宝源（推荐国内使用） 
nrm use taobao 
# 切换回官方源 
nrm use npm

测试源的响应速度

nrm test

自定义源操作

# 添加自定义源
nrm add <url>
nrm del <url>
