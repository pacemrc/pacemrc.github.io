安装Oracle

安装 .Net Framework 3.5

安装Oracle

解压压缩安装包，进入database文件夹，双击setup.exe运行安装程序

配置安全更新

去掉勾选接收安全更新，点击下一步

安装选项

选择“配置和安装数据库”，点击下一步

系统类

选择“服务器类“，点击下一步

网格安装选择

选择”单实例数据库安装“，点击下一步

安装类型

选择”高级安装“，点击下一步

产品语言

默认即可，点击下一步

数据库版本

选择”企业版“，点击下一步

安装位置

指定oracle的安装目录，此处我自定义了oracle11g目录，点击下一步

配置类型

选择”一般用途/事务类型“，点击下一步

数据库标识符

默认即可，点击下一步

配置选项

需要更改的项为【字符集】，选择”使用Unicode“，点击下一步

管理选项

默认即可，点击下一步

数据库存储

默认即可，点击下一步

备份和恢复

自行选择是否需要备份，此处默认即可，点击下一步

方案口令

可选择所有账户使用相同的口令，123.com，点击下一步

概要

建议保存响应文件，方便后续查看，最后点击完成

安装产品

oracle开始安装

配置

打开cmd命令行

sqlplus / as sysdba

创建表

#创建临时表空间
CREATE TEMPORARY TABLESPACE firsttmpspacea TEMPFILE 'C:\oracle11g\firsttmpspacea.dbf' SIZE 50M;

#创建表空间
CREATE TABLESPACE firstspace DATAFILE 'C:\oracle11g\firstspace.dbf' SIZE 100M;

#创建数据库

#创建用户
CREATE USER user01 IDENTIFIED BY user01 DEFAULT TABLESPACE firstspace TEMPORARY TABLESPACE firsttmpspacea;

#授权数据库连接
grant dba,connect to user01;

基础使用

#查看数据库名称
SELECT name FROM v$database;

**#**查看所有表
****SELECT table_name FROM all_tables;

#查看所有数据库
SELECT instance_name, status FROM v$instance;
