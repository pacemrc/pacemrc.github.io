Docker安装

Docker

RPM包安装

卸载旧版本

yum remove docker docker-client docker-client-latest docker-common docker-latest docker-latest-logrotate docker-logrotate docker-engine

安装依赖

yum install -y yum-utils device-mapper-persistent-data lvm2

设置镜像源

yum-config-manager --add-repo https://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

安装

yum makecache fast
yum install docker-ce docker-ce-cli containerd.io

启动

systemctl start docker
systemctl enable docker

二进制安装

下载地址：https://download.docker.com/linux/static/stable/x86_64/docker-23.0.2.tgz

安装

mkdir -p /usr/local/docker/data /usr/local/docker/exec /usr/local/docker/log /usr/local/docker/config
tar -xf docker-23.0.2.tgz -C /usr/local/docker
mv /usr/local/docker/docker /usr/local/docker/bin


echo "export PATH=/usr/local/docker/bin:$PATH" >> /etc/profile
echo "export DOCKER_HOST=unix:///usr/local/docker/exec/docker.sock" >> /etc/profile
source /etc/profile

创建配置文件

cat << EOF >> /usr/local/docker/config/daemon.json
{
    "data-root": "/usr/local/docker/data",
    "exec-root": "/usr/local/docker/exec",
    "hosts": [
      "unix:///usr/local/docker/exec/docker.sock",
      "tcp://0.0.0.0:2375"
    ],
    "pidfile": "/usr/local/docker/exec/docker.pid",
    "registry-mirrors": [
        "https://docker.1ms.run",
        "https://doublezonline.cloud",
        "https://dislabaiot.xyz",
        "https://docker.fxxk.dedyn.io",
        "https://dockerpull.org",
        "https://docker.unsee.tech",
        "https://hub.rat.dev",
        "https://docker.1panel.live",
        "https://docker.nastool.de",
        "https://docker.zhai.cm",
        "https://docker.5z5f.com",
        "https://a.ussh.net",
        "https://docker.udayun.com",
        "https://hub.geekery.cn"
    ]
}
EOF

启动守护进程

dockerd --config-file=/usr/local/docker/config/daemon.json

即可使用docker

Docker-compose

curl -L https://github.com/docker/compose/releases/download/v2.16.0/docker-compose-`uname -s`-`uname -m` -o /usr/local/bin/docker-compose

chmod +x /usr/local/bin/docker-compose
