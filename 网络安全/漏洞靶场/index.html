DVWA

切换网站环境

切换默认网站的PHP命令行版本和网站的程序语言版本为php7.x版本

创建数据库

下载项目

项目地址：https://github.com/digininja/DVWA

将下载的DVWA-master.zip文件保存至/www/admin/localhost_80/wwwroot目录下，解压并目录重命名为dvwa

创建配置文件

cd /www/admin/localhost_80/wwwroot/dvwa/config/
cp config.inc.php.dist config.inc.php

配置数据库连接

vim config.inc.php

===确保数据库连接信息正确===

修改php配置

vim /usr/local/phpstudy/soft/php/php-5.5.38/etc/php.ini

======将以下配置修改为On=====
display_errors = On
display_startup_errors = On
allow_url_fopen = On
allow_url_include = On

配置目录写权限

chmod 777 /www/admin/localhost_80/wwwroot/dvwa/hackable/uploads/
chmod 777 /www/admin/localhost_80/wwwroot/dvwa/config

ll /www/admin/localhost_80/wwwroot/dvwa/hackable/uploads/
ll /www/admin/localhost_80/wwwroot/dvwa/config

重启中间件

重启实际使用的中间件，如apache，nginx

安装检查

http://ip/dvwa/setup.php

确保所有项状态都是绿色即可。图中reCAPTCHA key: Missing可忽略，也可参考https://blog.csdn.net/WZZ_2363173126/article/details/120707038解决。

安装

安装检查没问题后，点击下方的Create/Reset Database按钮导入数据库文件，出现图示结果即安装成功。

登录地址

http://ip/dvwa/login.php
admin / password

Sqli-labs

创建数据库

无需另外创建数据库，后续会自动创建

下载项目

项目地址：https://github.com/Audi-1/sqli-labs

将下载的sqli-labs-master.zip文件保存至/www/admin/localhost_80/wwwroot目录下，解压并目录重命名为sqli

配置数据库连接

vim /www/admin/localhost_80/wwwroot/sqli/sql-connections/db-creds.inc

===确保数据库连接信息正确，只需配置root用户密码即可===

切换网站环境

sqlilab某些函数不支持php7.x版本，需安装php5.x版本。切换默认网站的PHP命令行版本和网站的程序语言版本为php5.x版本

重启中间件

略

访问

http://ip/sqli

点击Setup/reset Database for labs 进行导入数据库文件，出现图示界面即安装成功
