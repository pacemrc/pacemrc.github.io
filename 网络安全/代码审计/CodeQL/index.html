在VSCode中安装CodeQL

前言

安装环境

macOS Sonoma 14.4 M3芯片
Visual Studio Code 1.100.3
codeql-cli 2.21.3
codeql-lib 2.21.3

软件地址

介绍：包含CodeQL的标准库和内置的query文件
地址：https://github.com/github/codeql

介绍：CodeQL的二进制程序
地址：https://github.com/github/codeql-cli-binaries/

安装

对于M芯片的MacOS系统，需要安装Xcode和Rosetta 2以兼容CodeQL

安装Xcode

方式一：
在App Store搜索Xcode进行安装，安装的Xcode一般为最新版本。Xcode软件对MacOS的系统版本有要求，若macos版本为较低版本提示无法安装，可使用方式二安装。

方式二：
根据MacOS的系统版本选择兼容的Xcode版本，访问https://developer.apple.com/cn/support/xcode/查看对应的版本
如MacOS 14.5可选择Xcode 15.4版本下载，访问该链接进入软件包的下载：https://developer.apple.com/download/all/?q=xcode
或者修改下方URI的版本号直接下载安装包
https://download.developer.apple.com/Developer_Tools/Xcode_15.4/Xcode_15.4.xip

安装Rosetta 2

softwareupdate --install-rosetta

安装codeql-cli

下载安装

MacOS系统选择codeql-osx64.zip文件下载

https://github.com/github/codeql-cli-binaries/releases/download/v2.21.3/codeql-osx64.zip
将下载的codeql-osx64.zip文件解压并重命名为codel-cli

配置环境变量

macOS默认使用的shell是zsh，修改全局配置文件默认为~/.zprofile

vim ~/.zprofile

export CODEQL_HOME=/Users/pacemrc/software/CodeQL/codeql-cli
export PATH="${CODEQL_HOME}:${PATH}"

source ~/.zprofile

验证安装

codeql version

安装codeql-lib

下载压缩包

下载的clodeql-lib需要和codeql-cli版本保持一致，此处为2.21.3版本

访问https://github.com/github/codeql进行下载
https://github.com/github/codeql/archive/refs/tags/codeql-cli/v2.21.3.zip
下载完成后将codeql-codeql-cli-v2.21.3.zip文件解压缩并重命名为codeql-lib

安装VSCode

安装步骤略

安装CodeQL插件

在VSCode的插件商店中搜索CodeQL，安装插件

安装CodeQL插件后，需要在该插件中配置codeql-cli的可执行文件地址

在CLI扩展路径中填写codeql-cli目录中codeql可执行文件的绝对路径

至此，安装结束

快速使用

创建CodeQL数据库

将源码项目编译为CodeQL数据库，需要根据源代码项目安装对应的编程语言环境以及相关依赖包管理工具。如java-sec-code项目由java开发，使用的包管理工具为maven，那么需要提前安装java和maven软件并配置好环境变量

codeql database create /Users/pacemrc/software/CodeQL/codeql-db/java-sec-code-db --language=java --command="mvn clean package -DskipTests" --source-root=/Users/pacemrc/IdeaProjects/java-sec-code-master

编译后的数据库如下，其中src.zip文件为编译前的源代码项目

打开CodeQL库工作区

在VSCode中将codeql-lib目录作为工作空间打开

选择codeql-lib文件夹

进入CodeQL插件面板

CODEQL面板中有LANGUAGE、和QUERIES三个选项卡。

在LANGUAGE选项卡中选择需要扫描项目的开发语言，如Java

在DATABASES选项卡中添加创建的CodeQL数据库

在QUERIES选项卡中运行ql规则文件扫描

运行CodeQL查询

如Java-sec-code项目，语言选择Java，在数据库中选择编译后的数据库文件夹，选择完该两项后，在QUERIES中会自动显示可以执行的QL

以内置的java->ql->src->Security->CWE->CWE-089->SqlTained.ql规则文件为例，在左侧点击规则文件右侧的三角形箭头即可运行该条ql规则

点击运行后，右下角出现扫描进度条

扫描完成后，在右侧自动弹出扫描结果面板

点击蓝色超链接字体可查看源代码文件的详细内容
