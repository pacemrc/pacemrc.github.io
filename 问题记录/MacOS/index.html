安装软件后打开提示应用已损坏

在 macOS 上，出现“应用已损坏”的提示通常是因为安全设置阻止了未签名的应用程序。你可以尝试以下步骤来解决这个问题：

解决方法

  1. 使用终端解除隔离：
    运行以下命令来解除应用的隔离：

    xattr -rd com.apple.quarantine "/Applications/QQ.app"

  2. 确认执行权限：
    确保应用程序和 ChromeDriver 都有正确的执行权限：

    chmod +x "/Users/pacemrc/tmp/chrome_test/chrome-mac-arm64/Google Chrome for Testing.app/Contents/MacOS/Google Chrome for Testing"
chmod +x /path/to/chromedriver

  3. 系统偏好设置：

    • 打开“系统偏好设置” > “安全性与隐私”。
    • 在“通用”选项卡中，如果看到“仍要打开”按钮，点击它以允许应用运行。
