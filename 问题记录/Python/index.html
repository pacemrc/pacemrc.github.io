离线安装第三方库

下载whl包

该命令会将目标包及依赖包下载保存，但不安装

python -m pip download 包名==版本号

示例：

python -m pip download requests==2.26.0

安装whl文件

将安装包上传至需要离线安装的机器上，执行命令安装单个包，需要将所有包安装

python -m pip install --no-index --no-deps 包文件名

参考

https://blog.csdn.net/S_o_l_o_n/article/details/110492806

https://www.coder.work/article/7534231

生成项目依赖文件

安装pipreqs库

pip install pipreqs

生成requiremenet文件

在项目根目录下执行以下命令，会在项目目录下生成requirements.txt文件

pipreqs ./ --encoding=utf8  --force

Python程序打包

pyinstaller是一个第三方包，可将python程序的所有依赖打包为一个程序

安装

Linux

yum -y install python3
yum -y install glibc binutils

pip3 install wheel
pip3 install --upgrade pip
pip3 install pyinstaller

Windows

pip3 install pyinstaller

https://pyinstaller.org/en/stable/installation.html

使用

pyinstaller -F pyfile.py

将在当前目录的dist目录下生成一个二进制文件，在Windows系统上生成exe文件，在Linux系统上生成ELF文件

循环导入

当A模块需要从B模块导入函数，B模块也需要从A模块中导入函数时，此时会报错循环导入

test1.py

from test2 import word2

word1="Hello World"
print(word2)

test2.py

from test1 import word1

word2="Hello World"
print(word1)

此时运行test1或test2将会循环导入异常

注意：python在导入模块时，会自动执行目标模块中的所有语句。

压缩JSON数据

将python对象转换为字符串数据

json.dumps(dict_data_info, separators=(',', ':'), ensure_ascii=False)

示例

从文本文件中将字符串加载为JSON格式数据并压缩数据

import json

with open('file/jsonlog.txt', 'r', encoding='utf-8') as f:
    line = f.readline()
dict_data = json.loads(line)
json_data = json.dumps(dict_data, separators=(',', ':'), ensure_ascii=False)
print(json_data)
