JavaFx教程

概述

JavaFx

javafx可以开发图形化工具，在jdk8以前javafx内置在jdk中，在jdk8以后javafx独立成开源项目。

SDK下载：https://gluonhq.com/products/javafx/

教程：https://jenkov.com/tutorials/javafx/index.html https://openjfx.cn/index.html

SceneBuilder

scene builder可以帮助java开发者以图形化的界面设计界面。

软件下载：https://gluonhq.com/products/scene-builder/

理论

javafx程序运行需要窗口，窗口之上需要设置舞台，舞台上面需要设置布局，布局里面可以添加控件

核心

Stage

Stage是java程序的窗口。

创建窗口

Application类的start方法会自动创建一个stage

Stage myStage = new Stage();

显示窗口

myStage.show();

窗口设置舞台

VBox vBox = new VBox(new Label("javafx"));
Scene myScene = new Scene(vBox);
Stage myStage = new Stage();
myStage.setScene(myScene);

窗口设置标题

stage.setTitle("this is title");

设置窗口位置

myStage.setX(0);
myStage.setY(0);

设置窗口宽高

myStage.setWidth(500);
myStage.setHeight(300);

设置窗口模态

窗口模态是指当父窗口打开子窗口后，操作父窗口是否会阻塞

Modality.APPLICATION_MODAL 当窗口设置为APPLICATION_MODAL模态，无法操作其他窗口
Modality.WINDOW_MODAL 当窗口设置为WINDOW_MODAL模态，可以操作其他窗口
Modality.NONE

public class DemoApplication extends Application {
    @Override
    public void start(Stage primaryStage) throws Exception {

        primaryStage.setTitle("JavaFX App");

        Stage stage = new Stage();
        stage.initModality(Modality.APPLICATION_MODAL);
        //stage.initModality(Modality.WINDOW_MODAL);
        //stage.initModality(Modality.NONE);

        primaryStage.show();
        stage.showAndWait();

    }
    
    public static void main(String[] args) {
        launch();
    }
}

设置窗口拥有者

public static void main(String[] args) {
    launch(args);

@Override
public void start(Stage primaryStage) {
    primaryStage.setTitle("JavaFX App");

    Stage stage = new Stage();
    stage.initModality(Modality.WINDOW_MODAL);

    stage.initOwner(primaryStage);

    primaryStage.show();
    stage.showAndWait();
}

窗口生命周期事件

Close Stage Event Listener

监听点击窗口的X动作，执行自定义功能

stage.show();
stage.setOnCloseRequest((event) -> {
    System.out.println("Closing Stage");
});

Hiding Stage Event Listener

该监听在窗口被隐藏前调用

primaryStage.setOnHiding((event) -> {
    System.out.println("Hiding Stage");
});

Hidden Stage Event Listener

该监听在窗口被隐藏后调用

primaryStage.setOnHidden((event) -> {
    System.out.println("Stage hidden");
});

Showing Stage Event Listener

该监听在执行show方法之前调用

primaryStage.setOnShowing((event) -> {
    System.out.println("Showing Stage");
});

Stage Keyboard Events

该监听键盘操作，下面示例，当按下ESC键时关闭窗口，按下回车键时窗口放大一倍

stage.addEventHandler(KeyEvent.KEY_PRESSED,  (event) -> {
    System.out.println("Key pressed: " + event.toString());

    switch(event.getCode().getCode()) {
        case 27 : { // 27 = ESC key
            stage.close();
            break;
        }
        case 10 : { // 10 = Return
            stage.setWidth( stage.getWidth() * 2);
        }
        default:  {
            System.out.println("Unrecognized key");
        }
    }
});

Scene

scene表示窗口的场景，javafx中一切GUI组件都必须要放在scene上面。一个场景一次只能添加到一个窗口上，且一个窗口一次只能显示一个场景

创建场景

VBox  vBox  = new VBox();
Scene scene = new Scene(vBox);

将场景设置到窗口

VBox vBox = new VBox();
Scene scene = new Scene(vBox);

Stage myStage = new Stage();
myStage.setScene(scene);
myStage.show();

场景图

所有视觉组件（控件、布局等）都必须附加到要显示的场景，并且该场景必须附加到舞台才能使整个场景可见。附加到场景的所有控件、布局等的总对象图称为场景图。

Node

添加到场景图的所有组件都称为节点。每个JavaFX节点（子类）实例只能添加到JavaFX场景图中一次。换句话说，每个节点实例只能出现在场景图中的一个位置。

Parent

继承了Parent类的子类可以拥有子节点。有以下类Group、Region和WebView。

Group

JavaFX Group 组件是一个容器组件，对其子组件不采用特殊布局。所有子组件（节点）的位置都是 0,0。JavaFX Group 组件通常用于对一组控件（作为一个组）整体应用某种效果或变换。如果需要对组内部的子控件进行布局，可将它们嵌套在布局组件中，然后将布局组件添加到组中。

Region

javafx.scene.layout.Region类是所有布局类的基类

WebView

Properties

JavaFX 属性是 JavaFX 控件的一种特殊成员变量。JavaFX 属性通常用于包含控件属性，如 X 和 Y 位置、宽度和高度、文本、子控件以及 JavaFX 控件的其他核心属性。您可以为 JavaFX 属性附加更改监听器，这样当属性值发生变化时，其他组件就会收到通知，您还可以将属性相互绑定，这样当一个属性值发生变化时，另一个也会发生变化。

下拉框-ComboBox

创建下拉框

ComboBox comboBox = new ComboBox();

下拉框添加选项

ComboBox<String> stringComboBox = new ComboBox<>();
stringComboBox.getItems().add("aaa");
stringComboBox.getItems().add("bbb");
stringComboBox.getItems().add("ccc");

下拉框到场景中

@Override
public void start(Stage stage) {

    ComboBox<String> stringComboBox = new ComboBox<>();
    stringComboBox.getItems().add("aaa");
    stringComboBox.getItems().add("bbb");
    stringComboBox.getItems().add("ccc");

    VBox vBox = new VBox();
    vBox.getChildren().add(stringComboBox);

    Scene scene = new Scene(vBox);
    stage.setTitle("JavaFX App");
    stage.setScene(scene);
    stage.setWidth(500);
    stage.setHeight(300);
    stage.show();
}

设置下拉框可编辑

按钮-Button

按钮监听事件

@Override
public void start(Stage stage) {

    Label label1 = new Label("Not Click!");
    Button button = new Button("Click");

    button.setOnAction(actionEvent -> {
        label1.setText("Clicked!");
    });

    VBox vBox = new VBox();
    vBox.getChildren().addAll(label1,button);

    Scene scene = new Scene(vBox);
    stage.setTitle("JavaFX App");
    stage.setScene(scene);
    stage.setWidth(300);
    stage.setHeight(100);
    stage.show();
}
