jsp运行过程

请求JSP的流程大致如下：在第一次请求相应的JSP时，会将jsp进行一轮jsp语法的解析，解析主要是从jsp的语法解析到java语法，在默认为work的目录下生成相应的java文件(内容本质是一个Servlet)，之后在对java文件进行编译，生成对应的class文件并且加载到JVM中。

jsp指令

JSP指令用来设置整个JSP页面相关的属性。

page指令

功能：

Page指令为容器提供当前页面的使用说明。一个JSP页面可以包含多个page指令。

语法：

<%@ page attribute="value" %>
等价xml语法：
<jsp:directive.page attribute="value" />

常用属性：

import:导入要使用的Java类
language:定义JSP页面所用的脚本语言，默认是Java
contentType:指定当前JSP页面的MIME类型和字符编码
pageEncoding:jsp页面编码，默认是UTF-8

include指令

功能：

JSP可以通过include指令来包含其他文件。被包含的文件可以是JSP文件、HTML文件或文本文件。包含的文件就好像是该JSP文件的一部分，会被同时编译执行。

语法：

<%@ include file="文件相对 url 地址" %>
等价XML语法：
<jsp:directive.include file="文件相对 url 地址" />

taglib指令

功能：

JSP API允许用户自定义标签，一个自定义标签库就是自定义标签的集合。
Taglib指令引入一个自定义标签集合的定义，包括库路径、自定义标签。

语法：

<%@ taglib prefix="c" uri="<http://java.sun.com/jsp/jstl/core>" %>
等价XML语法：
<jsp:directive.taglib prefix="c" uri="<http://java.sun.com/jsp/jstl/core>" />

使用：
<c:out value="test"/>

jsp语法

<% %>：脚本片段，可在该片段定义java代码
<%! %>：声明片段，可在该片段中定义变量供脚本片段中引用
<%= %>：表达式，可引用脚本变量中的值

jsp的EL表达式

通过EL表达式可以在jsp页面中获取来自4个作用域application、session、request和pageContext中的数据

启用EL表达式

<%@ page isELIgnored="false" %>

jsp隐式对象

request

javax.servlet.http.HttpServletRequest类的实例，通过setAttribute()方法将对象存储在session作用域

常用方法：

request.getRequestURL().toString()   //获取请求的URL

//获取项目的URL地址
<%
    String contextPath = request.getContextPath();
    StringBuffer requestURL = request.getRequestURL();
    String projectUrl = requestURL.substring(0, requestURL.indexOf(request.getRequestURI())) + contextPath;
%>

response

javax.servlet.http.HttpServletResponse类的实例

out

java.io.PrintWriter类的实例

常用方法：

out.print(); 在页面中输出文本，不换行。
out.println(); 在页面中输出文本并换行。
out.flush();  刷新输出流。

jspContext

对象 JspContext类的实例

application

javax.servlet.ServletContext 类的实例

可在servlet的doGet/doPOST方法中通过req.getServletContext()获取，通过setAttribute()方法将对象存储在application作用域

config

javax.servlet.ServletConfig 类的实例

pageContext

javax.servlet.jsp.PageContext 类的实例

session

javax.servlet.http.HttpSession 类的实例

可在servlet的doGet/doPOST方法中通过req.getSession()获取，通过setAttribute()方法将对象存储在session作用域

page

对象 Object类的实例

单个u
