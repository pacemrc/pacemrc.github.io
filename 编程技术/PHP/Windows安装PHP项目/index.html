Windows安装PHP项目

下载地址

php：https://windows.php.net/download

Microsoft Visual C++运行库:

https://learn.microsoft.com/zh-CN/cpp/windows/latest-supported-vc-redist?view=msvc-170

composer： https://getcomposer.org/download/

php包： https://packagist.org/

本文档示例安装php7.4.33

安装PHP

下载安装包

访问https://windows.php.net/download

php–7.4需要VC15运行库，需提前安装

php分为线程安全、非线程安全包，此处选择VC15 x64 Non Thread Safe，下载zip文件

将下载的php-7.4.33-nts-Win32-vc15-x64.zip文件解压至特定目录

配置系统环境变量

*\PHP_HOME：xxxx*

*\Path：%PHP_HOME%*

验证安装

*\php -v*

安装composer

访问https://getcomposer.org/download/下载Composer-Setup.exe

手动指定php目录下的php.exe

点击Install进行安装，安装完成后需运行一下composer。打开cmd执行composer命令即可

下载composer.phar

https://getcomposer.org/download/下拉出现composer.phar 下载链接，选择较新的一个即可

将下载的composer.phar移动至php安装目录，同时创建一个composer.cmd文件，写入以下内容

@php “%~dp0composer.phar” %*

验证安装

composer –version

设置阿里云镜像

composer config -g repo.packagist composer https://mirrors.aliyun.com/composer/

查看composer配置

composer config -l -g

开启常用扩展

编辑php安装目录php.ini文件

将extension前面的“;”号去掉保存即可

安装php项目

以安装thinkphp为例

创建一个空的文件夹

访问https://packagist.org/，搜索thinkphp

进入topthink/think

命令：

安装最新版本：composer create-project topthink/think

安装指定版本：composer create-project topthink/think tp 5.1.41

进入到空文件夹，打开cmd，执行命令即可安装
