ECMAScript

ECMAScript是一种脚本程序设计语言的标准化规范，由 Ecma 国际组织制定。JavaScript是这种规范的实现和扩展。

历史发展

自1997年发布第一版以来，ECMAScript 经历了多次更新。最著名的是 ECMAScript 5（ES5，2009年发布）和 ECMAScript 6（ES6，2015年发布）。

主要特性

ES5 特性

  • 严格模式（Strict Mode）：提供更严格的错误检查，帮助避免常见错误。
  • JSON 支持：原生支持 JSON 解析和序列化。
  • 函数增强：如 Function.prototype.bind。
  • 数组方法：如 forEach、map、filter、reduce 等。
  • 对象属性控制：通过 Object.defineProperty 等方法更细粒度控制对象属性。

ES6 特性

  • 类（Classes）和模块（Modules）。
  • 箭头函数（Arrow Functions）：简化函数定义的语法。
  • 模板字符串（Template Strings）：用于字符串插值。
  • 解构赋值（Destructuring Assignment）。
  • Promises：用于异步编程。
  • 增强的对象字面量（Enhanced Object Literals）。
  • 块级作用域构造（如 let 和 const 关键字）。
  • 迭代器（Iterators）和生成器（Generators）。
  • 新的数据结构，如 Map、Set、WeakMap 和 WeakSet。

后续版本（ES2016及以后）

  • 异步函数（Async Functions）。
  • 展开操作符（Spread Operator）和剩余参数（Rest Parameters）。
  • 新的内置方法和小的语言改进。

变量

变量的声明


const：声明一个块作用域的只读常量。

let

声明一个块作用于的局部变量，可选初始化一个值。使用范围：块级作用域。如if判断语句、for循环体内

{
    let a = 10;
    var b = 1;
}

console.log(a);  //ReferenceError: a is not defined
console.log(b);  //1




for (let i = 0; i < 5; i++) {
    console.log(i);   //  0 1 2 3 4 
}

console.log(i);   //ReferenceError: i is not defined

var

声明一个变量，可选初始化一个值。 使用范围：函数作用域和全局作用域。

var和let的区别

  • var声明的变量可在声明前使用，值为undefined
  • let声明的变量续在声明后使用，否则会报错
console.log(a);   //undefined
var a = 1;

console.log(b);  //ReferenceError: Cannot access 'b' before initialization
let b =1;

Promise对象

“I Promise a Result!”

“Producing code（生产代码）” 是需要一些时间的代码

“Consuming code（消费代码）” 是必须等待结果的代码

Promise 是一个 JavaScript 对象，它链接生成代码和消费代码

语法

let myPromise = new Promise(function(myResolve, myReject) {
// "Producing Code"（可能需要一些时间）

  myResolve(); // 成功时
  myReject();  // 出错时
});

// "Consuming Code" （必须等待一个兑现的承诺）
myPromise.then(
  function(value) { /* 成功时的代码 */ },
  function(error) { /* 出错时的代码 */ }
);
let myPromise = new Promise(function(myResolve, myReject) {
    setTimeout(function() { myResolve("I love You !!"); }, 3000);
});

myPromise.then(function(value) {
    console.log(value);
});
