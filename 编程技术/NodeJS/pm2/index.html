PM2

pm2是专为NodeJS设计的进程管理工具。全称Process Manage 2，第二代进程管理工具。

官方文档：https://pm2.keymetrics.io/docs/usage/quick-start/

安装

npm install pm2 -g

常用命令

启动：pm2 start app.js
停止：pm2 stop app.js
删除：pm2 delete <name>|<id>
开机自启：pm2 startup
重启：pm2 restart <app>|<app1> <app2> <app3>|all

查看状态：pm2 list
查看日志：pm2 logs <name>|<id>
热重载：pm2 reload <name>|<id>

通过配置文件管理：
pm2 init
pm2 start/stop ecosystem.config.js


监控：
pm2 monit

查看应用元数据：
pm2 show api
