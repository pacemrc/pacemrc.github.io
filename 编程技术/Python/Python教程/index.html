Python常用模块

argparse

参考：https://docs.python.org/zh-cn/3/library/argparse.html#

命令行参数选项

使用步骤

创建解析器对象

parser = argparse.ArgumentParser()

常用参数：

  • prog：程序的名称。默认值：os.path.basename(sys.argv[0])
  • usage：描述程序用途的字符串。默认从添加到解析器的参数生成。
  • description：程序的描述信息。在选项参数的描述信息之前。
  • epilog：程序的额外描述信息。在选项参数的描述信息之后。

添加参数

parser.add_argument()
add_argument_group()添加多个参数组

add_argument()常用参数

  • name or flags：设置选项的名字，可以是单个字符(-f)，也可以是一个字符串列表(-f, –file)

  • action：指定选项参数如何处理，默认是store动作

    常用值：
store: 默认的action值，将参数值存储起来供后续使用。
store_true: 将True值存储起来，通常用于表示启用一个开关选项。
store_false: 将False值存储起来，通常用于表示禁用一个开关选项。**不为该参数指定值也可以运行

  • nargs：表示该选项值的个数，一个或多个，值为通配符 + ？等

  • default：选项没有指定值时，使用的默认值

  • type：参数值的数据类型，默认是字符串格式。

  • required：该选项参数是否为必须

  • help：为选项参数设置帮助文本信息

  • dest：用于指定参数值在args对象中的名字。默认情况下，argparse会基于参数名称作为该参数的名称

解析参数

parser.parse_args()

参数：

  • args：要解析的字符串列表。默认是从sys.argv获取。
  • namespace：用于获取属性的对象。默认值是一个新的Namespace对象

threading

功能：多线程执行任务

单线程

创建线程对象

t = threading.Thread()

语法：
t = threading.Thread(target,(args|kwargs))

参数：
target:需要运行的目标方法
name:线程的名字
args:使用元组方式向目标方法传递参数，单个参数表示，(param1,)，多个参数表示，(param1,param2,param3)
kwargs:使用字典的方式向目标方法传递参数，示例 {"key1": "value1","key2": "value2"}
daemon:守护程序，设置当主进程执行完成后，线程立即结束，值为布尔值

启动线程

t.start()

进程阻塞

含义：当线程任务运行完成后才继续执行主进程的任务

t.join()

多线程

定义一个线程列表

threads = []

创建多个线程对象并启动

for i in range(n):

t = threading.Thread()

t.start()

进程阻塞

for thread in threads:

t.join()

示例

--------------code-------------------
import time
import threading

def sing(n):
    print("I'm singing.")
    time.sleep(n)

def dance():
    print("I'm dancing.")

threads = []

for i in range(3):
    t = threading.Thread(target=sing,args=(3,))
    t.start()
    threads.append(t)

#等待所有线程执行完成后继续后续的任务
for thread in threads:
    thread.join()

dance()

--------------output-----------------

I'm singing.
I'm singing.
I'm singing.
I'm dancing.
